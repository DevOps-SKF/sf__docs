Установка:

python -m pip install "pelican[markdown]"

Создать директорию, перейти в нее, и запустить

pelican-quickstart

Важные директории:

  • /pelican/content - файлы .md и прочие
  • /pelican/themes - темы оформелния
  • /pelican/output - формируется при запуске pelican content
  • /docs - содержимое этой диреткории показывается на статическом сайте github.io. В реальности /pelican/output - линк на /docs.

https://devops-skf.github.io/sf__docs/ - "стандартное" обращение к статическим страницам.
